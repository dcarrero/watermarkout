## الميزات

- **السحب والإفلات**، النقر أو Ctrl+V لتحميل صورة
- **كشف تلقائي** لعلامات Gemini المائية (تحليل المكونات المتصلة + درجة الثقة)
- **تحديد يدوي** — ارسم مستطيلاً فوق أي علامة مائية
- **إصلاح خوارزمي** (أخذ عينات قائمة على الرقع مع المتوسط المرجح)
- **شريط قبل/بعد** لمقارنة النتائج
- **التحكم في التنعيم** (٠-٥ تكرارات غاوسية)
- **تحميل** كـ PNG نظيف
- **١٢ لغة** · **متجاوب** · **سهل الوصول**
- بدون أي تبعيات — HTML5 + Canvas API + Vanilla JS
