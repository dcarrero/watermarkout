# WatermarkOut v0.5

Verwijder watermerken van AI-gegenereerde afbeeldingen. **100% offline** — je afbeelding verlaat nooit je browser.

## Functies

- **Slepen en neerzetten**, klikken of Ctrl+V om een afbeelding te laden
- **Auto-detectie** van Gemini-watermerken (analyse van verbonden componenten + betrouwbaarheidsscore)
- **Handmatige selectie** — teken een rechthoek over elk watermerk
- **Algoritmische inpainting** (patch-gebaseerde sampling met gewogen gemiddelde)
- **Voor/na-slider** om resultaten te vergelijken
- **Verzachtingsregelaar** (0-5 Gaussiaanse iteraties)
- **Download** als schone PNG
- **12 talen** · **Responsive** · **Toegankelijk**
- Geen afhankelijkheden — HTML5 + Canvas API + Vanilla JS

## Snel starten

```bash
python3 -m http.server 8080
# Open http://localhost:8080
```
