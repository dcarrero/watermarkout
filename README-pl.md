# WatermarkOut v0.5

Usuń znaki wodne z obrazów wygenerowanych przez AI. **100% offline** — Twój obraz nigdy nie opuszcza przeglądarki.

## Funkcje

- **Przeciągnij i upuść**, kliknij lub Ctrl+V, aby załadować obraz
- **Auto-wykrywanie** znaków wodnych Gemini (analiza połączonych komponentów + wynik pewności)
- **Ręczny wybór** — narysuj prostokąt na dowolnym znaku wodnym
- **Algorytmiczny inpainting** (próbkowanie oparte na patchach ze średnią ważoną)
- **Suwak przed/po** do porównywania wyników
- **Kontrola wygładzania** (0-5 iteracji Gaussowskich)
- **Pobierz** jako czysty PNG
- **12 języków** · **Responsywny** · **Dostępny**
- Zero zależności — HTML5 + Canvas API + Vanilla JS

## Szybki start

```bash
python3 -m http.server 8080
# Otwórz http://localhost:8080
```
